# Fix the Group Stage tiebreak procedure

Group Stage ties follow the [Squash Canada singles round-robin procedure](https://squash.ca/wp-content/uploads/2011/07/Tie-Breaking-Procedures-for-Round-Robin-Draws-v3-as-of-January-16-2020.pdf): two tied Players are separated head-to-head; three or more are compared using Matches among the tied Players, then Games won, Game differential, and point differential, returning to head-to-head whenever only two remain. If Players remain statistically inseparable, an authorized Official Tournament Organizer or the Social Tournament creator orders them manually without a required explanation, while the system records the deciding person and time. Wildcard Qualifiers from unequal Groups are compared separately by Match win percentage, Game win percentage, and point win percentage before the same Organizer Tiebreak Decision; these fixed procedures avoid hidden ID-based ordering and unfair raw totals.
